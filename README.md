# 🎉 Xplore App – College Event Management System

A centralized platform to manage **college clubs, events, registrations, and announcements** with role-based access.

---

## 🚀 Features

### 👨‍💼 Admin Panel

* Create & manage events
* Approve/reject club activities
* View analytics & participation
* Send announcements to students

### 🎓 Student Panel

* Browse upcoming events
* Register for events
* Get real-time updates
* View personal event history

---

## 🧠 System Design

* Role-based architecture (Admin / Student)
* Scalable state management using **BLoC**
* REST API-based backend (in progress)
* SQL database for structured storage

---

## 🛠️ Tech Stack

* **Frontend:** Flutter, Dart
* **State Management:** BLoC
* **Backend:** REST APIs (in progress)
* **Database:** SQL
